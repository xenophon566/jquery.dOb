    Getting Started

    Dropdown menu for date choosing. As following : 

    $("#myBirthDate").dOb({
    dOb_Y: '#dOb_Y', // element year
    dOb_M: '#dOb_M', // element month
    dOb_D: '#dOb_D', // element day  
    from_Y: 1911, //from year(default: 1911)
    to_Y: 2013, //to year(default: 2013)
    isPadding: true //padding zero(default: true)
});

    Document

    HTML as following :
    <div id="myBirthDate">

    <select id="dOb_Y"><option value="1982">1982</option></select>

    <select id="dOb_M"><option value="3">03</option></select>

    <select id="dOb_D"><option value="23">23</option></select>
