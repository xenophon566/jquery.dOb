Getting Started

dOb.js is a dropdown menu to choosing your date of birthday. It getting start as following : 

$("#myBirthDate").dOb({
    from_Y: 1911,    //from year(default: 1911)
    to_Y: 6,     //to year(default: 10)
    digi: 2      //display digits(default: 2)
});

$("element").dOb();

Getting start the plugin


from_Y: 1911 (starting from 1911)

From which year


to_Y: 6 (to 6 years ago of this year)

To which year


digi: 2 (least 2 digits display)

Padding zero in front of digits


Document

HTML format as following :
<div id="myBirthDate">

<select id="dOb_Y"><option value="1982">1982</option></select>

<select id="dOb_M"><option value="3">03</option></select>

<select id="dOb_D"><option value="23">23</option></select>

</div>
  • Select id ( dOb_Y, dOb_M and dOb_D ) can not be changed.
  • The default value ( Year, Month and Day ) is fill in the field directly.
  • If the default value is unnormal, auto convert the date of the first set.
  • You can set non-numeric to default. ex."----" or blank.
