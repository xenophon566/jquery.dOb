Getting Started

dOb.js is a dropdown menu to choosing your date of birthday. It getting start as following : 

$("#myBirthDate").dOb({
	from_Y: 1911,	//from year(default: 1911)
	to_Y: 6, 	//to year(default: 10)
	digi: 2		//display digits(default: 2)
});

Getting start the plugin

$("element").dOb();

Calculate from which year

from_Y: 1911 ( start from 1911 )

Calculate to which year

to_Y: 6 ( ended in 6 years ago of this year )

Padding zero in front of digits

digi: 2 ( display least 2 digits )

Document

HTML format as following :

<div id="myBirthDate"> <select id="dOb_Y"><option value="1982">1982</option></select> <select id="dOb_M"><option value="3">03</option></select> <select id="dOb_D"><option value="23">23</option></select> </div>

No.1 Select id ( dOb_Y, dOb_M and dOb_D ) can not be changed.

No.2 The default ( Year, Month and Day ) is set directly in value field.

No.3 If the default value unnormal then auto setting the date of the first set.

No.4 We can setting non-numeric to default. ex."----" or blank.
