Getting Started

Dropdown menu calendar. Getting start as following : $("element").dOb();

$("#myBirthDate").dOb({
 from_Y: 1911, // from which year(default: 1911)
 to_Y: 2013, // to which year(default: 2013)
 isPadding: true // is padding zero(default: true)
});

Note

  1. Default values on select option's value directly.
  2. any type values ex. 2013, "----" or blank.
